# Tales of the Cake üç∞

A beautiful, responsive, and premium bakery website built with **React**, **Vite**, and **Tailwind CSS**.  
This website features a modern design, parallax scrolling, featured cake sliders, testimonials, and is fully mobile-friendly.

---

## Features

- Responsive layout for **mobile, tablet, and desktop**.  
- **Parallax scrolling** for a premium feel.  
- **Featured cake sliders** to showcase products.  
- **Testimonials** section for customer reviews.  
- Separate pages for **Home, About, Cakes, Contact** and more.  
- Easy to **customize** and **expand** in the future.  
- Tailwind CSS for fast styling and elegant design.
